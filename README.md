Summing products of multivariate polynomials and exponentials over tuples of naturals---
all source code is in [`src/PolyGeo.hs`](https://github.com/rafi230144/polygeo/blob/main/src/PolyGeo.hs).

To enter with `cabal repl` requires GHC version `9.14` and Cabal version `>= 3.0`;
if you don't have and don't want to install these, then first run `nix develop`
(assuming you have `nix`...).
